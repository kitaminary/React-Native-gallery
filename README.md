# React-Native-gallery
[DEMO]()

## Start

1. npm install --global expo-cli
2. npm install
3. expo start

## Used
1. React Native version 0.68.2
2. Node v14.19.3
3. HTML
4. CSS
5. JavaScript
6. Adaptable crossbrowser project

## Functionality
1. Photos with name and author name
2. Open/Close big photo by using TouchableHighlight
